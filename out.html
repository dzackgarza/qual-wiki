2.1.1 Superlinear Decay

:::{.exercise title="\(1/x^{4} + 1\), balancing exponentials"}
\begin{align*}\ \\int\_\\RR {1 \\over x\^4 + 1} = {\\pi \\sqrt{2} \\over 2} = {\\pi \\over \\sqrt 2}\ .\end{align*}

#completed

:::

:::{.solution}
The integrand \(f \in \mathsf{O}\text{\textbackslash qty}\frac{1}{z^{4}} \subseteq \mathsf{O}\text{\textbackslash qty}\frac{1}{z^{1 + \varepsilon}}\), so a semicircular contour works.
Factoring the denominator: find a principal root
\begin{align*}\ \\omega\^4 = -1 = e\^{i\\pi} \\implies \\omega = e{i\\pi\\over 4}\ ,\end{align*}
so
\begin{align*}\ z\^4 + 1 = (z-\\omega)(z-\\omega\\zeta_4)(z-\\omega\\zeta_4\^2)(z-\\omega\\zeta_4\^3) \\leadsto {\\pi \\over 4}, {3\\pi \\over 4}, {5\\pi \\over 4}, {7\\pi \\over 4}\ .\end{align*}
So there are two simple poles in \(\mathbb{H}\):

Write them as \(z_{1} = e^{\frac{i\pi}{4}}\) and \(z_{2} = e^{\frac{3i\pi}{4}}\).
Computing the residues:
\begin{align*}\ \\Res*{z=z_k} f(z)\ &= \\lim*{z\\to z*k} {z-z_k \\over z\^4 + 1} \\\ &\\eqLH \\lim*{z\\to z*k} {1\\over 4z\^3} \\\ &= {4z\^{-3} }\\evalfrom*{z=z*k}\ ,\end{align*}
so
\begin{align*}\ 2\\pi i \\sum \\Res*{z=z_k}\ &= {2\\pi i \\over 4}\\qty{z_1\^{-3} + z_2\^{-3}} \\\ &= {\\pi i \\over 2}\\qty{e\^{-3 i \\pi \\over 4} + e\^{-9 i \\pi \\over 4}} \\\ &= {\\pi i \\over 2}\\qty{e\^{-3 i \\pi \\over 4} + e\^{-i \\pi \\over 4}} \\\ &= {\\pi i \\over 2}\\cdot e\^{-2i\\pi \\over 4}\\qty{e\^{- i \\pi \\over 4} + e\^{i \\pi \\over 4}} \\\ &= \\pi i, {1\\over i} \\cos\\qty{\\pi \\over 4} \\\ &= {\\pi \\sqrt 2 \\over 2}\ .\end{align*}

:::

:::{.exercise title="\(1/x^{4} + 1\), half-line"}
\begin{align*}\ I \\da \\int_0\^\\infty {1\\over x\^4 + 1 }\\dx = {\\pi \\over 2\\sqrt 2}\ .\end{align*}

#completed

:::

:::{.solution title="Integrand symmetry"}
Note that the function is even, so
\begin{align*}\ \\int*{0\^\\infty} f(x) \\dx = {1\\over 2} \\int*\\RR f(x)\\dx = {1\\over 2} {\\pi \\over \\sqrt 2} = {\\pi \\over 2 \\sqrt 2}\ ,\end{align*}
using the solution from a previous problem.
:::

:::{.solution title="Sector"}
A sector will work, since there is a symmetry under \(z \rightarrow \zeta_{4}z\) and \(f(z) \sim z^{- 4}\), so the semicircular piece will vanish.
Take the contour \(\Gamma\) comprised of

  • \(\gamma_{1}:\left\{ {t + 0i{\ |\ }t \in \lbrack 0,R\rbrack} \right\}\),
  • \(C_{R}:\left\{ {Re^{it}{\ |\ }t \in \lbrack 0,\pi/2\rbrack} \right\}\),
  • \(\gamma_{2}:\left\{ {0 + it{\ |\ }t \in \lbrack 0,R\rbrack} \right\}\),

oriented counter-clockwise.
Note that \(z^{4} + 1 = \prod\limits_{k = 0}^{3}(z - \omega\zeta_{4})\) where \(\omega = e^{\frac{i\pi}{4}}\) and \(\zeta_{4} = e^{\frac{2\pi i}{4}} = i\), so there is only one pole at \(z_{0}{: =}e^{\frac{i\pi}{4}}\) within this contour.

Computing the symmetry:
\begin{align*}\ \\int*{\\gamma_2} f(z) \\dz\ &= \\int_R\^0 {i \\over (ti)\^4 + 1}\\dt \\qquad z=ti,, \\dz = i\\dt \\\ &= -i \\int_0\^R {1\\over t\^4 + 1}\\dt \\\ &= -i \\int*{\\gamma*1} f(z) \\dz\ ,\end{align*}
so applying the residue theorem and noting that $\int{CR}f\to 0$,
\begin{align*}\ 2\\pi \\Res*{z=z*0} f(z) = \\int*\\Gamma f(z) \\dz = \\qty{\\int*{\\gamma_1} + \\int*{C*R} + \\int*{\\gamma*2}}f \\too (1-i) I\ .\end{align*}
Computing the residue:
\begin{align*}\ \\Res*{z=z*0}f(z)\ &= {1\\over 4z\^3}\\evalfrom*{z= e\^{i\\pi \\over 4}} \\\ &= {1\\over 4}e\^{-3i\\pi \\over 4} \\\ &= {1\\over 4}e\^{5i\\pi \\over 4}\ .\end{align*}

Thus
\begin{align*}\ I\ &= (1-i)\\inv 2\\pi i \\Res\_{z=z_0} f(z) \\\ &= \\qty{\\bar{1-i} \\over \\abs{1-i}\^2} 2\\pi i \\cdot {1\\over 4}e\^{5i\\pi \\over 4} \\\ &= {1+i \\over 2} {\\pi i \\over 2}e\^{5i\\pi \\over 4} \\\ &= {e\^{i\\pi \\over 4} \\over \\sqrt{2} } {\\pi i \\over 2}e\^{5i\\pi \\over 4} \\\ &= {\\pi i \\over 2\\sqrt{2}}e\^{6i\\pi \\over 4} \\\ &= {\\pi i \\over 2\\sqrt{2}}(-i) \\\ &= {\\pi \\over 2\\sqrt{2}}\ .\end{align*}

:::

:::{.solution title="The log trick"}
Consider the auxiliary function \(g(z){: =}\log(z)f(z)\), and take a keyhole contour:

Let \(\Gamma\) be the counterclockwise contour consisting of

  • \(C_{\varepsilon} = \left\{ {\varepsilon e^{it}{\ |\ }t \in \lbrack 0 + \varepsilon,2\pi - \varepsilon\rbrack} \right\}\)
  • \(\gamma_{+} = \left\{ {x + i\varepsilon{\ |\ }x \in \lbrack\varepsilon,R\rbrack} \right\}\)
  • \(C_{R} = \left\{ {Re^{it}{\ |\ }t \in \lbrack 0 + \varepsilon,2\pi - \varepsilon\rbrack} \right\}\)
  • \(\gamma_{-} = \left\{ {x - i\varepsilon{\ |\ }x \in \lbrack\varepsilon,R\rbrack} \right\}\)

Computing the symmetry:
\begin{align*}\ \\int*{\\gamma*-}{\\log(z) \\over z\^{4} + 1} \\dz\ &= \\int*{R}\^\\eps {\\log(x-i\\eps) \\over (x-i\\eps)\^4 + 1} \\dx \\qquad z=x-i\\eps, \\dz = \\dx \\\ &\\to - \\int*{\\eps}\^R {\\log(x) + 2\\pi i\\over x\^4 + 1}\\dz \\\ &= -\\int*{\\gamma*+} {\\log(z) \\over z\^4+1}\\dz - 2\\pi i\\int*{\\gamma*+} {1\\over z\^4+1} \\dz\ ,\end{align*}
so
\begin{align*}\ \\int*{\\gamma*+} f + \\int*{\\gamma*-}f \\too -2\\pi i I\ .\end{align*}
By the ML estimate, since \(\log(z)/z^{4} \rightarrow 0\) as \(\text{\textbackslash abs}z \rightarrow \infty\), \(\int_{C_{R}}g(z) \rightarrow 0\).
Similarly, since \(\log(z)/(z^{4} + 1) \rightarrow 0\) as \(\text{\textbackslash abs}z \rightarrow 0\), \(\int_{C_{\varepsilon}} \rightarrow 0\).
We're then left with the sum of residues at \(e^{\frac{ki\pi}{4}}\) for \(k = 1,3,5,7\).
We have
\begin{align*}\ \\Res*{z=z_k} f(z)\ &= {1\\over 4z\^3}\\evalfrom*{z=z*k} \\\ &= {z\\over 4z\^4}\\evalfrom*{z=z*k} \\\ &= - {z\\over 4}\\evalfrom*{z=z*k} \\qquad \\text{ since } z_k\^4 = -1 \\\ &= -{z_k \\over 4}\ ,\end{align*}
so
\begin{align*}\ \\Res*{z=z_k}g(z) = -{z_k \\over 4}\\log(z_k)\ .\end{align*}

Now use that in the limit,
\begin{align*}\ 2\\pi i \\sum*k \\Res*{z=z*k}g(z)\ &= \\int*\\Gamma g(z) \\dz \\\ &= \\qty{\\int*{\\gamma*+} + \\int*{\\gamma_i}} f \\\ &= -2\\pi i I\ ,\end{align*}
so $I = -\sum_k \Res*{z=z_k} g(z)$.

Being very careful to note that we've chosen a branch of \(\log\) where \({Arg}(z) \in (0,2\pi)\) in order to get the signs right,
\begin{align*}\ -\\sum_k \\Res(z=z_k) g(z)\ &= {1\\over 4}\\qty{ z_1\\log(z_1) + z_2\\log(z_2)+z_3\\log(z_3)+z_4\\log(z_4) } \\\ &= {1\\over 4}\\qty{ z_1 {i\\pi \\over 4} + z_2 {3i\\pi \\over 4} + z_3 {5i\\pi \\over 4}+ z_4 {7i\\pi \\over 4} } \\\ &= {i\\pi \\over 16}\\qty{ z_1 + 3z_2 + 5z_3+ 7z_4 } \\\ &= {i\\pi \\over 16}\\qty{ \\omega \\zeta_4\^0 + 3\\omega\\zeta_4\^1 + 5\\omega\\zeta_4\^2+ 7\\omega \\zeta_4\^3 } \\\ &= {i\\pi \\omega \\over 16}\\qty{ 1 + 3i + -5 + -7i } \\\ &= {i\\pi \\omega \\over 16}(-4-4i) \\\ &= -{i\\pi \\omega \\over 4}(1+i) \\\ &= -{i\\pi \\omega \\over 2}{ \\omega \\over \\sqrt{2} } \\\ &= -i\\omega\^2 {\\pi \\over 2 \\sqrt 2} \\\ &= {\\pi \\over 2\\sqrt{2} }\ ,\end{align*}
using that \(\frac{\sqrt{2}}{2}(1 + i) = e^{\frac{i\pi}{4}} = \omega\) and \(\omega^{2} = i\).

:::{.exercise title="\(1/(1 + x^{2})^{2}\)"}
\begin{align*}\ \\int\_\\RR {1 \\over (1+x\^2)\^2} = {\\pi \\over 2}\ .\end{align*}

#completed

:::

:::{.solution}
Use that \(f(z) \sim 1/z^{4}\):

Then
\begin{align*}\ \\int*{C_1 + C_R} f = \\int*{C*1} f + \\int*{C*R} f = 2\\pi i \\Res*{z=i} {1\\over (1+z\^2)\^2}\ .\end{align*}

Note \(f\) factors:
\begin{align*}\ f(z) = {1 \\over \\qty{(z-i)(z+i)}\^2 }\ ,\end{align*}
so \(z = i\) is a pole of order 2.

Compute the residue within the contour:
\begin{align*}\ \\Res*{z=i} f(z)\ = \\lim*{z\\to i} \\dd{}{z} {1\\over (z+i)\^2}\ = - {2 \\over (z+i)\^3 }\\evalfrom\_{z=i}\ = - {2\\over (2i)\^3 }\ = {1\\over 4i} = -{i\\over 4}\ .\end{align*}

Now solve:
\begin{align*}\ 2\\pi i \\qty{- {i\\over 4}} = \\int*{C_1}f + \\int*{C*R} f \\da I_R + \\int*{C*R}f \\implies I_R = {\\pi \\over 4} - \\int*{C_R} f\ .\end{align*}

Note \(I_{R}\overset{R \rightarrow \infty}{\longrightarrow}I{: =}\int_{\mathbb{R}}f\), so it suffices to show
the semicircular error term vanishes as \(R \rightarrow \infty\).
Parameterize \(C_{R} = \left\{ {z = Re^{it}{\ |\ }t \in \lbrack 0,\pi\rbrack} \right\}\), so \(\text{\textbackslash abs}z = R\) on \(C_{R}\).
Letting \(z = e^{it}\), \(\, dz = iRe^{it}\, dt\),
\begin{align*}\ \\abs{\\int*{C_R} f(z) \\dz }\ &= \\abs{\\int*{C*R} {1\\over (1+z\^2)\^2} \\dz } \\\ &\\leq \\int*{C*R}\\abs{ {1\\over (1+z\^2)\^2} } \\dz \\\ &= \\int*{C*R} {1\\over \\abs{1+z\^2}\^2 } \\dz\\\ &\\leq \\int*{C*R} {1\\over \\qty{1 - \\abs{z}\^2 }\^2 } \\dz \\\ &= \\int*{C*R} {1\\over \\qty{1 - R\^2 }\^2 } \\dz \\\ &= {1\\over \\qty{1 - R\^2 }\^2 } \\int*{C_R} \\dz\\\ &= {1\\over \\qty{1 - R\^2 }\^2 } \\cdot \\length(C_R) \\\ &= {1\\over \\qty{1 - R\^2 }\^2 } \\cdot \\qty{1\\over 2}2\\pi R \\\ &= {\\pi R \\over R\^4 + O(R\^3)} \\\ &= \\pi\\qty{ 1 \\over R\^3 + O(R\^2)} \\\ &\\convergesto{R\\to\\infty} 0\ ,\end{align*}
where we've used a variant of the triangle inequality:
\begin{align*}\ \\abs{a\\pm b} \\geq \\abs{ \\abs{a} - \\abs{b} } \\implies {1\\over \\abs{a \\pm b}} \\leq {1\\over \\abs{a} - \\abs{b} }\ .\end{align*}

:::

\begin{align*}\ x\^2 + 4x + 13 = 0\ &\\implies x\^2 + 4x + \\qty{4\\over 2}\^2 = -13 + \\qty{4\\over 2}\^2 \\\ &\\implies (x+2)\^2 = -9 \\\ &\\implies x = -2 \\pm 3i\ ,\end{align*}

\begin{align*}\ z_1 \\da -2+3i && z_2 \\da -2 - 3i\ .\end{align*}

So let \(\Gamma\) be comprised of

  • \(C_{1} = \lbrack - R,R\rbrack\),
  • \(C_{2} = \left\{ {Re^{it}{\ |\ }t \in \lbrack 0,\pi\rbrack} \right\}\),

    \begin{align*}\ 2\\pi i \\sum*{z_k\\in \\HH} \\Res*{z=z*k}f(z) = \\int*\\Gamma f = \\qty{\\int*{C_1} + \\int*{C*2}}f\ ,\end{align*}
    where $\int{C2} f\to 0$ and $\int{C1} f\to I$.
    So in the limit, $I = 2\pi i \Res{z=z1} f(z)\(.Computingthisresidue:note\$ z_{1}\) is a pole of order 2, so

    so
    \begin{align*}\ I = 2\\pi i \\cdot {i\\over 54} = -{\\pi \\over 27}\ .\end{align*}

2.1.2 Linear or sublinear decay

:::{.exercise title="\(x\sin(x)/1 + x^{2}\)"}
\begin{align*}\ I = \\int\_\\RR {x\\sin(x) \\over 1 + x\^2}\\dx\ .\end{align*}

#completed

:::

:::{.solution}
Write \(f(z) = \frac{ze^{iz}}{1 + z^{2}}\), and note that \(f \in \mathsf{O}\text{\textbackslash qty}\frac{1}{z}\), so the usual semicircular contour with the ML estimate won't work.
Claim: a semicircular contour with a better estimate will work:

Writing \(f(z) = e^{iz}g(z)\) where \(g(z){: =}\frac{z}{1 + z^{2}}\), we have \(g \in \mathsf{O}\text{\textbackslash qty}\frac{1}{z} \rightarrow 0\) as \(\text{\textbackslash abs}z \rightarrow \infty\), so Jordan's lemma applies.
Write \(C_{1} = \lbrack - R,R\rbrack\) and \(C_{R} = \left\{ {Re^{it}{\ |\ }t \in \lbrack 0,\pi\rbrack} \right\}\), then
\begin{align*}\ \\abs{\\int*{C_R} e\^{iz} g(z)\\dz }\\leq \\pi M_R,, \\qquad M_R \\da \\sup*{z\\in C*R}\\abs{z\\over 1+z\^2}\ .\end{align*}
Now use that \(\frac{z + 1}{z^{2}} \leq M\text{\textbackslash abs}z\) for \(\text{\textbackslash abs}z\) large enough to conclude this integral goes to zero.
By the residue theorem,
\begin{align*}\ 2\\pi i \\sum*{z*k\\in \\HH}\\Res*{z=z*k}f(z) = \\int*{C*1 + C_R} f = \\qty{\\int*{C*1} + \\int*{C*R}}f \\converges{R\\to\\infty} \\int*{C*1} f = I\ ,\end{align*}
so it suffices to compute the residues of \(f\).
Check that \(1 + z^{2} = (1 + i)(1 - i)\), so \(z_{1} = i \in \mathbb{H}\) is a simple pole and
\begin{align*}\ 2\\pi i \\Res*{z=i} f(z)\ &= 2\\pi i \\lim\_{z\\to i} {e\^{iz} \\over z+i} \\\ &= 2\\pi i {i\\over 2ei} = {\\pi \\over e}\ ,\end{align*}
so
\begin{align*}\ I = {\\pi \\over e}\ .\end{align*}

:::{.exercise title="\(\cos(x)/i + x\)"}
\begin{align*}\ I \\da \\int\_\\RR {\\cos(x) \\over x+i}\\dx\ .\end{align*}

#completed

:::

:::{.solution}
Note that the usual thing won't work, since \(\frac{\cos(z)}{z + i} \neq \Re\text{\textbackslash qty}\frac{e^{iz}}{z + i};\) the complex constant in the denominator throws this off!
Instead, use \(\cos(z) = \frac{1}{2}(e^{iz} + e^{- iz})\) to decompose into two integrals:
\begin{align*}\ I \\da \\int*\\RR {\\cos(z) \\over z+i}\ = {1\\over 2} \\int*\\RR {e\^{iz} \\over z+i} + {1\\over 2}\\int*\\RR {e\^{-iz} \\over z+i} \\da \\int*\\RR f*1 + \\int*\\RR f_2\ ,\end{align*}

Write \(f(z){: =}\frac{z^{\frac{1}{3}}}{z^{2} + 1}\), the claim is that an indented semicircular contour will work:

Why:

Note the poles at \(z = \pm i\).

\begin{align*}\ \\Res\_{z=i} f(z) &= {i\^{1\\over 3} \\over 2i} = {1\\over 2e\^{i\\pi \\over 3}}\ .\end{align*}

By the residue theorem,
\begin{align*}\ 2\\pi i \\cdot\ {1\\over 2e\^{i\\pi \\over 3} }\ &=\ (1+e\^{i\\pi \\over 3})I \\ \\\ \\implies\ I\ &= {i\\pi \\over 2e\^{i\\pi \\over 3} (1+e\^{i\\pi \\over 3}) } \\\ &= {i\\pi \\over 2} \\qty{ e\^{i\\pi \\over 3} + e\^{2i\\pi \\over 3} }\\inv \\\ &= {i\\pi \\over 2} \\qty{ e\^{i\\omega} + e\^{2i\\omega} }\\inv,\\qquad \\omega={\\pi\\over 3} \\\ &= {i\\pi \\over 2} \\qty{ e\^{3i\\omega\\over 2} \\qty{ e\^{-i\\omega\\over 2} + e\^{i\\omega\\over 2}} }\\inv \\\ &= i\\pi e\^{-3i\\omega\\over 2}{1\\over \\cos\\qty{\\omega\\over 2}} \\\ &= i\\pi e\^{-i\\pi\\over 2 }{1\\over \\cos\\qty{\\pi \\over 6}} \\\ &= {\\pi \\over \\sqrt{3}}\ ,\end{align*}
where we've used the "exponential balancing trick" (see complex arithmetic section).

2.4.3 Square Roots / Slits

:::{.concept}
\envlist

  • The residue at \(\infty\): if \(\Gamma\) is a positively oriented curve (counterclockwise about \(z = 0\)), then
    \begin{align*}\ \\Res*{z=\\infty}f(z) = -{1\\over 2\\pi i}\\oint*\\Gamma f(z) \\dz,\\quad \\Res*{z=\\infty}f(z) = \\Res*{z=0} -{1\\over z\^2}f\\qty{1\\over z}\ .\end{align*}

    • Be sure to check to see if \(z = \infty\) is a branch point!
  • A standard argument: for \(f(z) = \sqrt{z}\), take a path \(\gamma(t) = e^{2\pi it}\) for \(t \in \lbrack 0,1\rbrack\).
    Then \(f(\gamma(0)) = f(z)\) and \(f(\gamma(1)) = \sqrt{e^{2\pi i}z} = e^{i\pi}\sqrt{z} = - \sqrt{z}\), so the monodromy/phase factor introduced by square roots is \(- 1\).

Note that \(f(x) = (x + 2)(x + 1)\), so the singularities on on \(\mathbb{R}_{< 0}\).
The function isn't even, so a semicircular contour won't work.
Attempting to find a ray-like symmetry only yields one option:
\begin{align*}\ f(z) = f(e\^{i\\theta } z) \\implies \\theta = 2k\\pi\ ,\end{align*}
which suggests a keyhole.
But since there's no \(\log\) in \(f\), there's no monodromy, and the contributions cancel out.
So introduce a log with a branch cut along \(\theta = 0\), and consider
\begin{align*}\ \\int f(z) \\log(z)\ .\end{align*}

Let \(\gamma_{+} = \left\{ {t + i\varepsilon{\ |\ }t \geq 0} \right\}\) (right-to left) and \(\gamma_{0} = \left\{ {t - i\varepsilon{\ |\ }t \geq 0} \right\}\) (left-to-right).
Now use the general fact
\begin{align*}\ \\int*{\\gamma*+}f(z)\\log(z) \\dz &= \\int*\\eps\^R f(t+i\\eps)\\log(t+i\\eps) \\dt \\too \\int*\\RR f(t)\\log(t)\\dt \\\ \\int*{\\gamma*-}f(z)\\log(z) \\dz &= \\int*R\^\\eps f(t-i\\eps)\\log(t-i\\eps) \\dt \\too -\\int*\\RR f(t)\\qty{ \\log(t) + 2\\pi i} \\dt \\\ ,\end{align*}
thus
\begin{align*}\ \\int*{\\gamma*+}f(z)\\log(z)\\dz + \\int*{\\gamma*-}f(z)\\log(z)\ &\\too \\int_0\^\\infty f(t)\\log(t)\\dt - \\int_0\^\\infty f(t)(\\log(t) + 2\\pi i) \\dt \\\ &= -2\\pi i \\int_0\^\\infty f(t) \\dt\ .\end{align*}

By the ML estimate, the semicircular piece vanishes.
Miraculously, since \(\lim\limits_{x \rightarrow 0}\frac{x\log(x)}{x^{n} + c} = 0\) for any \(c > 0\) and \(n \geq 1\), the inner indented pieces goes to zero.
Parameterize by \(z = \varepsilon e^{it}\)
\begin{align*}\ \\int*{C*\\eps} f(z)\\log(z)\\dz\ &\\approx \\int*{\\eps}\^{2\\pi - \\eps} {\\log(\\eps e\^{it}) \\over \\eps\^2 + 3\\eps + 2} \\eps e\^{it}\\dt \\\ &= \\int*{\\eps}\^{2\\pi - \\eps} {\\log(\\eps) + it \\over \\eps\^2 + 3\\eps + 2} \\eps e\^{it}\\dt \\\ &\\approx \\int*\\eps\^{2\\pi - \\eps} {\\eps \\log(\\eps) + c_1 \\over \\eps\^2 + c_2}\\dt \\\ &\\convergesto{\\eps\\to 0} 0\ ,\end{align*}
where I've been extremely sloppy and left out many negligible \(e^{it}\) terms.
By the residue theorem,
\begin{align*}\ 2\\pi i \\sum*{z*k\\in \\CC} \\Res*{z=z*k} f(z)\\log(z) = \\int*\\Gamma f(z)\\log(z)\\dz = -2\\pi i\\int*\\RR f(z)\\dz\ .\end{align*}
There are just two simple poles at \(z_{1} = - 1,z_{2} = - 2\), so
For \(z_{1}\):
\begin{align*}\ \\Res*{z=-1}{\\log(z) \\over (z+1)(z+2)}\ = \\lim*{z\\to -1} {\\log(z) \\over z+2}\ = \\log(-1)\ ,\end{align*}
and for \(z_{2}\),
\begin{align*}\ \\Res*{z=-2}{\\log(z) \\over (z+1)(z+2)}\ = \\lim\_{z\\to -2} {\\log(z) \\over z+1}\ = -\\log(-2)\ .\end{align*}

Thus
\begin{align*}\ I\ &= -(\\log(-1) - \\log(-2)) \\\ &= - (\\ln(1) + i\\pi) + (\\ln(2) + i\\pi) \\\ &= \\ln(2)\ ,\end{align*}
noting that we've chosen a branch of \(\log(z){: =}\ln\text{\textbackslash qty}{\text{\textbackslash abs}z} + i{Arg}(z)\) where \({Arg}(z) \in (0,2\pi)\).
